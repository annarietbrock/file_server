Rules

The game of P, R, S is based on trust, skill and determination. With each round, each player must choose whether to play:
  • Rock
  • Paper
  • Scissors
The winner of each round is determined by which player throws the "strongest" object. The strength of each object is as follows:
  • Rock beats Scissors
  • Paper beats Rock
  • Scissors beats Paper

If both players throw the same object, neither player gets a point.
